# StudentIDFinder

A mobile application for managing lost and found student identification cards.

## Technologies Used

- Android Studio
- Java
- XML
- Node.js
- Express.js
- MySQL
- Tesseract OCR
- Twilio SMS
- Git & GitHub

## Features

- Officer login
- Capture student ID using camera
- OCR extraction of registration number
- Student lookup
- Automatic SMS notification
- Collection tracking
